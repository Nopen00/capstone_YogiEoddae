# capstone_YogiEoddae
# 📍 Yogi-Eoddae (요기어때?)
**미디어 기반 여행 정보 서비스 (2026 Hanium Dream Up ICT Mentoring)**

## 🚀 프로젝트 소개
유튜브 여행 영상의 데이터를 분석하여 실제 장소 정보를 제공하고, 네이버 지도 API를 통해 사용자에게 최적화된 여행 코스를 제안하는 서비스입니다.

## 🛠 Tech Stack (Backend)
- **Language:** Python 3.x
- **Framework:** Django
- **Database:** MySQL
- **APIs:** 한국관광공사(KorService2), NAVER Maps, YouTube Data API

## 📂 담당 역할 (Lee Tae-yong)
- 백엔드 서버 구축 및 API 연동
- 데이터베이스 모델링 (ERD)
- 위치 기반 데이터 가공
